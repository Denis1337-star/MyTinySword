# MyTinySword

**MyTinySword** — это прототип 2D RTS-игры на Unity, созданный как портфолио-проект.  
В проекте реализованы базовые механики стратегии в реальном времени: управление рабочими, сбор ресурсов, взаимодействие с постройками, найм юнитов и игровое UI.

---

## Обзор проекта

Цель проекта — реализовать базовый игровой цикл RTS:

- нанять рабочего
- отправить его на добычу ресурса
- доставить ресурс на базу
- управлять юнитами и взаимодействовать с окружением

Проект создавался как практика в:
- архитектуре gameplay-систем
- работе с AI/логикой юнитов
- организации кода в Unity
- отладке состояний, анимаций и игровых взаимодействий

---

## Использованные технологии

- **Unity**
- **C#**
- **NavMeshPlus** — навигация для 2D-проекта
- **Cinemachine** — работа с камерой
- **Tilemap + Rule Tiles** — построение карты
- **Animator + Blend Tree** — анимация движения и состояний
- **UI System** — элементы игрового интерфейса

## Реализованные механики

### Геймплей
- найм рабочих
- выделение юнитов
- перемещение юнитов
- добыча ресурсов
- перенос ресурсов на базу
- взаимодействие с постройками

- работа камеры через Cinemachine

## Архитектура проекта

Проект разделён на несколько логических зон ответственности:

- **Системы ввода и выделения**  
  Отвечают за выбор юнитов, обработку кликов и команд игрока.

- **Логика юнитов**  
  Управляет поведением рабочих: движение, поиск цели, добыча, перенос ресурсов, смена состояний.

- **Логика построек**  
  Отвечает за найм рабочих и взаимодействие зданий с игровым процессом.

- **Система ресурсов**  
  Описывает добываемые объекты и процесс их сбора.

- **UI**  
  Показывает информацию об объектах и даёт игроку кнопки управления.

- **Анимация**  
  Переключает состояния персонажей через параметры Animator и Blend Tree.
